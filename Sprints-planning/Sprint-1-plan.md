## Project Overview
This project is a recipe management app.  
Users can view recipes, save their favorites, and add their own recipes.

The goal of Sprint 1 is to set up the project and build basic components.

## Sprint Goals
- Set up GitHub repository and branches
- Create a React + TypeScript project using Vite
- Add a README file
- Build basic recipe components
- Deploy the app using Vercel
